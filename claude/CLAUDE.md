# Global Claude Code Instructions

## Commit Convention
Use **Conventional Commits** by default: https://www.conventionalcommits.org

```
<type>(<optional scope>): <description>

<optional body>
```
Common types: `feat`, `fix`, `refactor`, `test`, `docs`, `chore`

Override only when:
- The repo's existing commits use a different convention (match what's there)
- I explicitly specify a different format in the plan or chat
